# theme_bootstrap_mosaic
TYPO3 compatible adaption of the "Mocsaic - Responsive Professional Theme" by simpleqode

# Features

* Isotope Effekt for EXT:news via sys_categories
* Drop Down Menus
* fullscreen width carousel
